# ПроФайно Icon Studio 💛💙

**ПроФайно Icon Studio** — українська міні майстерня для створення стильних іконок папок, ярликів і PNG/ICO значків прямо в браузері.

Проєкт зроблений для творчих людей, які хочуть швидко прикрасити свої папки, архіви, музичні проєкти, документи, медіафайли або робочий стіл без складних програм.

## Що вміє

- створювати сучасні іконки папок у різних стилях;
- додавати текст, символи, emoji та власні зображення;
- рухати елементи мишкою по папці;
- змінювати розмір, нахил, колір і вигляд елементів;
- приховувати або прибирати непотрібні елементи;
- використовувати багато готових сучасних значків;
- експортувати PNG у різних розмірах;
- створювати `.ico` для ярликів Windows;
- зберігати дизайн як проєкт і відкривати його пізніше;
- працювати локально, без сервера та складного встановлення.

## Рекомендовані розміри для Windows

Для ярликів і папок Windows найкраще використовувати `.ico`, який містить кілька розмірів:

| Розмір | Для чого підходить |
|---|---|
| 16×16 | дрібні списки у Провіднику |
| 32×32 | стандартні ярлики |
| 48×48 | великі значки |
| 64×64 | комфортний проміжний розмір |
| 128×128 | якісні великі значки |
| 256×256 | високоякісні значки для великих екранів |

## Як користуватися

1. Відкрийте `index.html` у браузері.
2. Оберіть стиль папки, колір, символи й напис.
3. Перетягуйте елементи мишкою, як вам зручно.
4. Експортуйте PNG або ICO.
5. Встановіть іконку для папки у Windows через властивості ярлика або папки.

## Онлайн через GitHub Pages

Після завантаження репозиторію на GitHub можна увімкнути GitHub Pages:

1. Відкрийте репозиторій.
2. Перейдіть у **Settings**.
3. Відкрийте **Pages**.
4. У полі **Source** виберіть **Deploy from a branch**.
5. Branch: `main`, folder: `/root`.
6. Натисніть **Save**.

Після цього GitHub дасть посилання на онлайн версію.
