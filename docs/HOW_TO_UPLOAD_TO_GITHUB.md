# Як завантажити ПроФайно Icon Studio на GitHub

## 1. Створіть репозиторій

1. Натисніть `+` у правому верхньому куті GitHub.
2. Оберіть `New repository`.
3. Назва: `profaino-icon-studio`.
4. Оберіть `Public`.
5. Натисніть `Create repository`.

## 2. Завантажте файли

1. Відкрийте створений репозиторій.
2. Натисніть `Add file`.
3. Оберіть `Upload files`.
4. Перетягніть усі файли з архіву.
5. Натисніть `Commit changes`.

## 3. Увімкніть GitHub Pages

1. Перейдіть у `Settings`.
2. Відкрийте `Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Натисніть `Save`.

Через деякий час зʼявиться посилання на сайт.
