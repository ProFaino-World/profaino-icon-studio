# Як встановити іконку для папки у Windows

## Для папки

1. Натисніть правою кнопкою на папку.
2. Оберіть `Властивості`.
3. Відкрийте вкладку `Налаштування`.
4. Натисніть `Змінити піктограму`.
5. Оберіть файл `.ico`, створений у ПроФайно Icon Studio.
6. Натисніть `OK` і `Застосувати`.

## Для ярлика

1. Натисніть правою кнопкою на ярлик.
2. Оберіть `Властивості`.
3. Натисніть `Змінити піктограму`.
4. Виберіть `.ico` файл.
5. Збережіть зміни.

## Порада

Для найкращої якості використовуйте `.ico` з розмірами 16, 32, 48, 64, 128 і 256 px.
